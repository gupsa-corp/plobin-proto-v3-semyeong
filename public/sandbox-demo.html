사용 방법

  • 페이지 로드 시 자동으로 서버에서 샌드박스 템플릿 목록을 가져옵니다
  • 템플릿 카드에서 "템플릿으로 생성" 버튼을 클릭하면 해당 템플릿을 선택할 수 있습니다
  • 새로고침 버튼으로 최신 템플릿 목록을 다시 불러올 수 있습니다
  • 상단의 "API 테스트" 버튼으로 직접 API 응답을 확인할 수 있습니다
